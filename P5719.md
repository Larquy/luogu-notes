# P5719 【深基4.例3】分类平均

## 题目描述

给定 $n$ 和 $k$，将从 1 到 $n$ 之间的所有正整数可以分为两类：A 类数可以被 $k$ 整除（也就是说是 $k$ 的倍数），而 B 类数不能。请输出这两类数的平均数，精确到小数点后 $1$ 位，用空格隔开。

数据保证两类数的个数都不会是 $0$。

## 输入格式

输入两个正整数 $n$ 与 $k$。

## 输出格式

输出一行，两个实数，分别表示 A 类数与 B 类数的平均数。精确到小数点后一位。

## 输入输出样例 #1

### 输入 #1

```
100 16
```

### 输出 #1

```
56.0 50.1
```

## 说明/提示

数据保证，$1 \leq n\leq 10000$，$1 \leq k \leq 100$。

## AC代码
~~~
#include <bits/stdc++.h>
using namespace std;

int main()
{
    int n,k;
    cin >> n >> k;
    int numA=0, numB=0,sumA=0,sumB=0;
    for (int i=1; i<=n; i++)
    {
        if (i%k == 0)
        {
            numA++;
            sumA+=i;
        }
        else
        {
            numB++;
            sumB+=i;
        }
    }
    double avA = (sumA * 1.0) / numA;
    double avB = (sumB * 1.0) / numB;
    
    cout << fixed << setprecision(1) << avA << " " << avB << endl;

    return 0;
}
~~~

## 总结
* C++如何限定输出精度？
fixed + setprecision
fixed: 使用定点表示法（固定小数点位置）
setprecision(n): 当和fixed一起使用时，n就代表小数点后要保留的位数
#include < iomanip >

  * 一旦设置了fixed或setprecision，它会一直影响之后所有的cout输出，直到之后再次修改格式。
  * 如果想要恢复默认的输出格式，可以使用cout.unsetf(ios::fixed); 来清除fixed标志
  * 这些设置只对（float、double）有效，对整数（int）没有影响。
